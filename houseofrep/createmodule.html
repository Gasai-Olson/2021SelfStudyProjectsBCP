Creating the file:

If you do not plan to use the template, create a new file, in your text editor of choice, with the extension ".OSDA"


MacOS Terminal Example:

touch examplefile. OSDA

Header:

Headers are encased by the "__BEGIN__" function, ex:

__BEGIN__{
}

To call a python files functions use the following code and replace "example.py" with your file:
__BEGIN__{
    Use example.py as obj
}

If your function needs an argument you can create a variable to store all input after function call:
__BEGIN__{
    Use example.py as obj
    x = INPUT.aftercall

}

Body:

The body stores all the functions and their triggers, the body is called with __FUNCTIONS__


the triggers and functions are stored in a dictionary in their respective order, (trigger:function)

Example Body:

__FUNCTIONS__ {
    calculate : obj.calculate(x);
    graph : obj.graph(x);
    example trigger: obj.example_function(x)
}

notice "obj." in the function values, this object corresponds to when we imported example.py as obj in our header
x also represents input after trigger from the header.
