About OSDA

OSDA or Open-Source Desktop Assistant is a modular open-source text interface. Modules are files that give more functionality to the text interface. For example, if you were to create a NFL module that recognizes player names you could ask the interface "how many catches does so-and-so have". Our modules are created for the users by the users and with such a simple process to make a module almost anyone with any programming experience can create a module. If your interested in creating modules please visit the "Create a Module" page to get started.
